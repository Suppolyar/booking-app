# EPN frontend on VUE 3 and TypeScript

## General
- Use npm
- use typescript
- Order tags in Vue files:
  - script
  - template
  - style
- Example vue file
```vue
<script setup lang="ts">
</script>

<template>
</template>

<style scoped>
</style>
```

## Getting Started
- `npm install`
- `npm run dev`

## Code Style
- We use eslint and airbnb rules

## Styles
- We use tailwind
- Don't write your own classes

## Directory Structure
- src
  - assets
  - pages
  - components
    - UI
  - types
  - services
  - usables
